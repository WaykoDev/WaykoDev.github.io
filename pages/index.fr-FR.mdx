# Introduction

Bienvenue à Nextra ! Ceci est un modèle de document de base. Vous pouvez l'utiliser comme point de départ pour votre propre projet :)

## Qu'est-ce que Nextra ?

Un framework de génération de sites **simple**, **puissant** et **flexible** avec tout ce que vous aimez de Next.js.

## Documentation

La documentation est disponible sur [https://nextra.site](https://nextra.site).

## Exemple d'intégration d'un composant ReactJS

import { useState } from "react";

{/* Import CSS modules */}
import styles from '../components/counters.module.css'

export const Counter = () => {
  const [count, setCount] = useState(0);
  return (
    <div>
      <button onClick={() => setCount(count + 1)} className={styles.counter}>
        Clicked {count} times
      </button>
    </div>
  );
};

<Counter />
